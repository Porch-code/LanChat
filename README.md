# LanChat

LanChat是基于Qt的局域网即时通信系统，系统采用C/S架构模型，客户端负责发起通信请求并接收服务器响应，服务器负责处理客户端请求并进行信息转发。通过TCP或UDP协议与服务器进行数据交换。选择Qt框架进行系统开发，因其具备跨平台支持，并且Qt提供了丰富的图形界面开发支持和网络通信模块。系统使用SQLite数据库存储结构化数据。
